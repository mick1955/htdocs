Bootstrap 5 has some contextual classes that can be used to provide "meaning through colors".

The classes for text colors are: .text-muted,
.text-primary, 
.text-success, 
.text-info, 
.text-warning, 
.text-danger, 
.text-secondary, 
.text-white, 
.text-dark, 
.text-body (default body color/often black) and 
.text-light
